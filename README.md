# Memorial-and-Comment-Dataset 清代奏折数据库

中文 | [English](README_EN.md)

我在北大国发院的读博生涯即将结束。

奏折数据是我在博士期间的核心研究对象。本质上，奏折背后的理论框架是agent和principal之间的信息传递和处理问题：皇帝和官员如何围绕信息进行互动，如何处理、报告和利用信息？我从24年开始收集并处理奏折和朱批数据，构建了一个清代奏折数据库，利用它们来回答前面的问题，并完成我的博士论文。令我愉快的是，组成博士论文的三章小论文是连贯的，均围绕着奏折展开，也堪堪算是研究了一个虽小但集中的问题。

但我对经济学学术感到非常厌倦。我会在半年内（26年底前）将这个数据库上传到这里，权当留作这几年的纪念，如果能帮到一些朋友也算没有浪费生命。
